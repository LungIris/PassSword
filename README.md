Titlu proiect: PassSword
Autor: Lung Iris Adriana

Aplicația PassSword este destinată gestionării securizate a autentificării.
Utilizatori își pot salva parolele personale, le pot gestiona cu ușurință, și pot analiza datele contului. PassSword folosește algoritmi avansați de securitate precum AES-256, PBKDF2 sau Bcrypt.

Link github: https://github.com/LungIris/PassSword

Pași de instalare și folosire:
    1. Clonarea repozitorului git în locația dorită de pe dispozitivul personal.
    2. Accesarea executabilului ”PassSword-1.0.0 Setup.exe”. Acest executabil va instala aplicația și va crea un shortcut pe desktop. 
    3. Accesarea aplicației din shortcut 
    4. Înregistrarea unui nou utilizator
    5. Autentificare

După urmarea acestor pași utilizatorul va avea acces la toate funcționalitățile aplicației.
